Implementation of the [W3C WebRTC API](https://www.w3.org/TR/webrtc/) in Elixir.

See `examples` directory for usage examples.

## Installation

Add `ex_webrtc` to dependencies in `mix.exs`

```elixir
def deps do
  [
    {:ex_webrtc, "~> 0.1.0"}
  ]
end
```
